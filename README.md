## 介绍
项目使用`djangorestframework`实现前后端分离

## 环境
python3.6+

mysql5.7+

## 环境搭建
1. 下载源码
    ```
    git clone git@gitee.com:qDonl/onlineShop.git
    ```

2. 进入项目
    ```
    cd onlineShop
    ```
   
3. 安装环境
    ```
    # path/to/onlineShop
    pip install pipenv
    pipenv install
    ```

4. 运行环境
    ```.env
    python manage.py runserver
    ```
